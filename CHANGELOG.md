# 更新日志 / Changelog

## v2.6.9 (2026-04-09)

### 新增功能

- **自动更新检测**
  - 启动后自动检查更新（优先 Gitee，失败则尝试 GitHub）
  - 支持手动检查更新按钮
  - 发现新版本时显示通知气泡，点击即可安装
  - 下载进度显示
  - 更新包自动解压和安装

- **导出调试信息**
  - 在设置页面新增"导出调试信息"按钮
  - 导出的调试信息包括：
    - 版本信息（当前版本、Godot 引擎版本）
    - 路径配置（可执行文件路径、游戏路径、保存路径）
    - 模组统计（总模组数、已启用模组数）
    - 最近 200 行日志内容

### 问题修复

- 修复更新脚本只复制文件不复制目录的问题
- 修复 PowerShell 窗口自动关闭导致程序退出的问题
- 修复硬编码子目录名称导致解压失败的问题
- 修复 OS.shell_open 打开脚本被识别为文本文件的问题

### 技术改进

- 使用动态子目录查找替代硬编码
- 使用 `start /B` 后台运行 PowerShell 更新脚本
- 更新脚本完成后自动退出，无需用户交互
