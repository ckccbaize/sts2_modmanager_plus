# 模组界面 UI 重构设计方案

**日期**: 2026-04-19
**状态**: 已批准
**版本**: v1.0

## 1. 设计概述

将模组管理器的模组界面从当前基础风格升级为 Steam × Windows 11 混合风格，包含 Tesla 风格拖拽启动按钮。

### 1.1 布局结构

```
┌─────────────────────────────────────────────────────────────┐
│ ⚔️ 杀戮尖塔2 模组管理器                    [操作按钮组]    │  ← 标题栏
├─────────────────────────────────────────────────────────────┤
│  [🔍 搜索模组...] [筛选 ▼] [全部][游戏性][美化]  [+安装]  │  ← 工具栏
├─────────────────────────────────────────────────────────────┤
│                    │  单机模组 │ 联机模组 │ 自定义预设    │  ← 启动模式
│   ┌─────────────┐  │───────────────────────────────────── │
│   │ 🎴 模组名称 │  │  模组名称              v1.0.0        │
│   │  作者 • 版本 │  │  ┌─游戏性─┐ ┌─兼容 v2.0+─┐          │
│   │         ⏽   │  │  作者: xxx │ 安装: 2024-01-01         │
│   ├─────────────┤  │                                       │
│   │ 🎴 模组名称 │  │  描述内容...                          │
│   │  作者 • 版本 │  │  ┌─ 📝 备注 ──────────────────┐      │
│   │         ⏽   │  │  │ 用户添加的备注内容...       │      │
│   └─────────────┘  │  └─────────────────────────────┘      │
│                    │                                       │
│                    │  [🚀 启用]  [🗑️ 卸载]                   │
├─────────────────────────────────────────────────────────────┤
│ 3 已启用 / 5 总数     [P]──[D]──[N]──[R]      ← 拖动启动  │  ← 底部栏
└─────────────────────────────────────────────────────────────┘
```

## 3. 组件规范

### 3.1 工具栏 (TopBar)

**结构**:
```
[HBoxContainer]
├── 搜索框 (药丸形 LineEdit)
├── 筛选下拉 (MenuButton)
├── 分类标签组 [全部][游戏性][美化]
└── 按钮组 [+安装][🔄刷新]
```

**样式**:
- 容器背景: `rgba(21, 31, 46, 0.5)` + `border: 1px solid rgba(86,131,172,0.15)` + `border-radius: 8px`
- 内边距: 10px 12px
- 间距: 10px

**搜索框**:
- 背景: `rgba(42, 71, 94, 0.6)` + `border: 1px solid rgba(86,131,172,0.3)`
- 圆角: 20px (药丸形)
- 宽度: 150-200px
- 聚焦: 边框变亮 `rgba(102,192,249,0.5)`

**分类标签**:
- 默认: 背景 `rgba(42, 71, 94, 0.4)`, 文字灰色
- 激活: 背景 `rgba(102,192,249,0.2)`, 边框 `#66c0f9`, 文字 `#66c0f9`

### 3.2 启动模式切换 (ModStartModes)

**结构**:
```
[HBoxContainer]
├── 单机模组
├── 联机模组
└── 自定义预设
```

**样式**:
- 高度: 40px (与内容区顶部对齐)
- 激活状态: `border-bottom: 2px solid #66c0f9`, 文字白色
- 未激活: 文字灰色
- 底部边框分隔

### 3.3 模组列表项 (ModItem)

**结构**:
```
[ModItem]
├── Bg (PanelContainer)
│   └── ItemMargin (MarginContainer, 14px)
│       └── HBox
│           ├── IconBg (40×40, 渐变背景)
│           ├── InfoVBox (名称 + 作者)
│           └── Toggle (36×20)
```

**样式**:
- 最小高度: 64px
- 背景默认: `rgba(255,255,255,0.03)` + `border-radius: 6px`
- 背景悬停: `rgba(255,255,255,0.06)`
- 背景选中: `rgba(102,192,249,0.15)` + 边框 `rgba(102,192,249,0.3)`
- 列表项间距: 6px

**图标**:
- 尺寸: 40×40px
- 圆角: 6px
- 游戏性: 渐变 `#f472b6` → `#be185d`
- 美化: 渐变 `#10b981` → `#047857`
- 默认: 渐变 `#66c0f9` → `#1b567a`

**Toggle 开关**:
- 尺寸: 36×20px
- 关闭: 背景 `rgba(42,71,94,0.8)`, 圆点白色偏左
- 开启: 背景 `#66c0f9`, 圆点白色偏右
- 动画: 0.2s ease

### 3.4 详情面板 (ModDetailsPanel)

**结构**:
```
[ScrollContainer]
└── DetailsPanelBg (PanelContainer)
    └── DetailsMargin (20px)
        └── DetailsVBox
            ├── DetailHeaderHBox (标题 + 版本)
            ├── HSeparator
            ├── BadgeHBox (徽章)
            ├── InfoLbl (作者/安装时间)
            ├── DescLbl (描述)
            ├── NotesBg (备注区)
            └── ActionHBox (按钮)
```

**样式**:
- 面板背景: `rgba(0,0,0,0.2)` + `border-radius: 8px` + `border: 1px solid rgba(86,131,172,0.15)`
- 标题: 22px, 字重 500, 颜色 `#66c0f9`
- 版本: 12px, 背景 `rgba(42,71,94,0.6)`, 圆角 4px

**徽章 (Badge)**:
- 圆角: 4px
- 内边距: 4px 10px
- 字号: 10px
- 游戏性: 渐变 `#f472b6` → `#be185d`
- 兼容性: 渐变 `#66c0f9` → `#1b567a`

**备注区**:
- 背景: `rgba(42, 71, 94, 0.15)`
- 边框: `1px solid rgba(86,131,172,0.15)`
- 圆角: 6px
- Header: 11px, 强调色, 大写

### 3.5 底部栏 (Footer)

**结构**:
```
[PanelContainer]
└── FooterHBox
    ├── StatusLabel (状态文字)
    ├── LaunchBarPlaceholder (Tesla启动按钮区域)
    └── StatusMini (提示文字)
```

**Tesla 启动按钮**:
- 布局: `[P]──[D]──[N]──[R]`
- 轨道宽度: 2px
- 档位间距: 30px
- 档位球尺寸: 28×26px

**档位状态**:
| 档位 | 颜色 | 功能 |
|------|------|------|
| P | #f472b6 (玫红) | 联机模式 |
| D | #66c0f9 (蓝色) | 模组模式 |
| N | #10b981 (绿色) | 空档/中立 |
| R | #f472b6 (玫红) | 原版模式 |

**档位球样式**:
- 背景: 渐变 `linear-gradient(145deg, #2a475e, #1b2838)`
- 边框: 2px solid (根据档位变色)
- 圆角: 12px
- 阴影: `0 2px 10px rgba(102,192,249,0.25)`

**拖拽交互**:
- 行程范围: -120px ~ +80px (从 N 档位开始)
- 吸附判定: 接近档位位置 ±15px 内
- 释放行为: 在档位上则启动，否则弹回 N
- 回弹动画: 0.3s cubic-bezier(0.175, 0.885, 0.32, 1.275)

## 6. 需要修改的文件

### 主要文件
- `modmanager.tscn` - 工具栏结构、启动模式tab、详情面板布局
- `modmanager.gd` - 样式初始化、事件处理逻辑
- `ui/mod_item.gd` - 列表项样式、选中状态、图标渐变

### 新建文件
- `ui/launch_bar.gd` - Tesla启动按钮组件
- `ui/launch_bar.tscn` - Tesla启动按钮场景
